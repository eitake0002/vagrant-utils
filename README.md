# vagrant-utils
vagrant command utilities

Version : 1.9.7 or later.

vagrantbox.es

http://www.vagrantbox.es/

vagrant official document

https://docs.vagrantup.com.

## Cheet Sheet

version check
```
vagrant -v
```

show help
```
vagrant -h
```

box help
```
vagrant box -h
```

add boxes
```
vagrant box add <name> <url>
```

list boxes
```
vagrant box list
```

remove box
```
vagrant box remove <box_name>
```
